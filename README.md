## 📖 Overview

The **Dual 555 Timer Traffic Light Circuit** is a discrete electronics project built using two NE555 timer ICs, capacitors, resistors, and LED sequencing logic to simulate a basic traffic light control system.

The project was hand-built on a breadboard using a schematic-based design workflow and demonstrates foundational electronics concepts including timer circuits, RC timing networks, LED sequencing, and breadboard prototyping.

## ⚙️ Development Stages

### Stage 1 — Schematic Analysis
Studied the original 555 timer traffic light schematic and identified RC timing sections and LED sequencing paths.

### Stage 2 — Breadboard Construction
Built the circuit physically using breadboard wiring and discrete components.

### Stage 3 — Timing Verification
Verified LED timing behavior and adjusted component placement and wiring connections.

### Final System
Completed working dual-555 traffic light simulator with automatic LED sequencing behavior.
