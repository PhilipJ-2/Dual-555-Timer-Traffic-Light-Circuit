# Breadboard Layout

This section documents the physical breadboard layout used for the traffic light circuit.

## Purpose

The breadboard layout was used to organize the two 555 timer ICs, timing components, and LED outputs.

## Layout Sections

- Power rails
- First 555 timer section
- Second 555 timer section
- RC timing components
- LED output section

## Notes

The circuit was tested on a breadboard to verify LED sequencing before considering any PCB or permanent version.
