# System Overview

This section documents the overall operation of the Dual 555 Timer Traffic Light Circuit.

## System Function

The system uses two 555 timer ICs to control traffic light style LED sequencing.

## Signal Flow

Power Supply  
→ 555 Timer Timing Network  
→ RC Charge/Discharge Behavior  
→ LED Output Control  
→ Traffic Light Sequence

## Notes

The project demonstrates how basic analog timer circuits can create automatic LED timing behavior without using a microcontroller.
