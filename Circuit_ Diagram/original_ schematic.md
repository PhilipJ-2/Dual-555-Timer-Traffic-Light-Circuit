# Original Schematic

This section documents the schematic used to build the dual 555 timer traffic light circuit.

## Purpose

The schematic shows how two NE555 timer ICs, resistors, capacitors, and LEDs are connected to create an automatic traffic light sequence.

## Main Components

- 2x NE555 timer ICs
- Red LED
- Yellow LED
- Green LED
- 100µF capacitors
- 100k resistor
- 47k resistor
- 330Ω resistors
- 180Ω resistor

## Notes

This circuit was built by following the schematic and translating the design onto a breadboard.
