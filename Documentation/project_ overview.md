# Project Overview

The Dual 555 Timer Traffic Light Circuit is a discrete electronics project designed to simulate a basic traffic light sequence using two NE555 timer ICs and RC timing networks.

## Project Goals

The project was developed to demonstrate:

- 555 timer circuit operation
- RC timing behavior
- LED sequencing logic
- Breadboard circuit construction
- Schematic-to-hardware implementation

## Main Subsystems

- Dual 555 timer stages
- RC timing networks
- LED output section
- Breadboard power distribution

## Purpose

This project demonstrates how timer ICs and passive components can create automatic sequencing behavior without using a microcontroller.
