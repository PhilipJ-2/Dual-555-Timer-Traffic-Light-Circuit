# Hardware Debugging

This section documents hardware testing and troubleshooting areas from the Dual 555 Timer Traffic Light Circuit project.

## Main Debugging Areas

- Breadboard wiring verification
- 555 timer IC orientation
- LED polarity checks
- Capacitor placement
- RC timing behavior
- Power rail stability

## Notes

The circuit required careful wiring verification because incorrect timer pin connections affected LED sequencing behavior.

Timing behavior was also checked to verify stable LED transitions and proper oscillation.
